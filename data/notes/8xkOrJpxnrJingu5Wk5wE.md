
To have the md preview export as html and the toc displayed add the folloing line to the Dendron notes

html:
  embed_local_images: true
  embed_svg: true
  offline: true
  toc: true
print_background: false
export_on_save:
  html: true
toc:
  depth_from: 1
  depth_to: 6
  ordered: false
---
